This tool counts the number of methods in the application, as there's a limit of 2^16 = 65536 methods. For more information please see:
https://developer.android.com/tools/building/multidex.html#about

This library is hosted on github: https://github.com/mihaip/dex-method-counts

HOW TO USE
-In Windows: "java -jar dex-method-counts.jar path/to/App.apk"
